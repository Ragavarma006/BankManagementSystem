# 🏦 Bank Management System

A console-based Bank Management System built in **C++** using Object Oriented Programming concepts.

## 📋 Features

- ✅ Create New Account
- ✅ Login System with Password
- ✅ Deposit Money
- ✅ Withdraw Money
- ✅ Check Balance
- ✅ View Account Details
- ✅ View All Accounts
- ✅ Delete Account
- ✅ File Handling (Data Persistence)
- ✅ Colored Terminal UI

## 🛠️ Technologies Used

- Language  : C++
- Concepts  : OOP, File Handling, Classes
- Compiler  : g++ (MinGW 13.2.0)
- OS        : Windows
- Editor    : VS Code
